# hivemind has moved

This project has been renamed to **devsper** and split into separate repositories:

| Repository | Description |
|---|---|
| [devsper-com/runtime](https://github.com/devsper-com/runtime) | Python runtime, Rust worker, docs, tests, examples |
| [devsper-com/registry](https://github.com/devsper-com/registry) | Go API, React web, deploy config, Terraform infra |
| [devsper-com/platform](https://github.com/devsper-com/platform) | Platform dashboard & API |

**PyPI:** `pip install devsper` (the `hivemind-ai` package name is deprecated)

**Docs:** [docs.devsper.com](https://docs.devsper.com)
